Instagram

Fecha de lanzamiento de la app

Creada por Kevin Systrom y Mike Krieger, fue lanzada el 6 de octubre de 2010.

Empresa encargada de su desarrollo

Meta Platforms, Inc.

Característica más atractiva/novedosa de la app.

Instagram es una red social principalmente visual, donde un usuario puede publicar fotos y videos de corta duración, aplicarles efectos y también interactuar con las publicaciones de otras personas, a través de comentarios y me gusta.

Tecnologías/herramientas implementadas en su desarrollo

  • Usan Django en Amazon High-CPU Extra Large
  • Usan como servidor WSGI a Gunicorn
  • La base de datos que se usa es PostgreSQL.
  • Para realizar las conexiones a la base de datos con PostgreSQL, principalmente desde Django se usa Pgbouncer.
  • para todo los que es contenido principal del feed de actividad y las notificaciones de Instagram se usa Redis.
  • Para el uso de la API de geo-búsqueda se usa Apache Solr que permite el consumo de datos por medio de JSON.
  • Para estos casos en los que se tiene que notificar a un usuario en tempo real de una nueva foto publicada se usa Gearman.
  • Usan Munin para revisar todos los parámetros gráficos de los sistemas.
  • También se usa Pingdom para la supervisión externa del servicio y PagerDuty para las incidencias y las notificaciones.
  • Y para el informe de errores usan Sentry, escrita usando Django por la gente de Disquis.

Youtube

Fecha de lanzamiento de la app

El dominio fue activado el 14 de febrero de 2005,​ y el 23 de abril fue cargado el primer vídeo, “Me at the Zoo” ('Yo en el zoológico') por el cofundador de la plataforma Jawed Karim.​

Empresa encargada de su desarrollo

Adquirida por Google, en 2006, por 1.65 millones de dólares.

Característica más atractiva/novedosa de la app.

YouTube es un sitio web de origen estadounidense dedicado a compartir videos. Presenta una variedad de clips de películas, programas de televisión y vídeos musicales, así como contenidos amateur como videoblogs y YouTube Gaming.

Tecnologías/herramientas implementadas en su desarrollo

  • El lenguaje de programación que usa es Python y C, donde Python permite un desarrollo más rápido y flexible.
  • Usa además Psyco que es un compilador C que se puede importar desde Python y que acelera notablemente las aplicaciones Python.
  • Usa MySQL de código abierto, donde se almacenan los nombres de usurarios, descripciones, tags, etc.
